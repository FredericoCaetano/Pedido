Pedido Especial Este é um programa Python que cria uma interface gráfica de usuário (GUI) usando a biblioteca Tkinter. O programa simula um pedido especial onde o usuário é convidado a decidir se deseja continuar o relacionamento.

Funcionalidades: Opção "Sim":

Ao clicar no botão "Sim", a interface se transforma para exibir uma nova mensagem e uma imagem de uma abelha. Além disso, uma janela adicional é aberta com uma animação de coração. Opção "Não":

Ao clicar no botão "Não", uma janela de aviso é exibida, informando que a opção é inválida. O usuário pode fechar essa janela ao clicar no botão "Fechar". Movimento do Botão "Não":

O botão "Não" pode ser movido aleatoriamente na janela principal ao passar o mouse sobre ele.

Pré-requisitos: Python 3.x instalado. Módulos necessários: tkinter, pillow (PIL), ttk.

Como Executar: Certifique-se de ter os pré-requisitos instalados. Execute o script pedido_especial.py.
